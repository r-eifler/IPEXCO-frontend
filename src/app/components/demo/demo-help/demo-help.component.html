HOW TO: Compute Plans

Compute Plan

To compute a new plan, you have to click the New Plan button in the navigation bar.

This opens a new view on the right. Here you can select the goals for the plan. In the top you can see the global goals. These goals have to be satisfied by all plans and are automatically included.

You start the plan computation with the button in the bottom right corner. To compute the plan can take some time.

When the computation finished, a new plan will appear in the navigation bar. The number in the orange circle indicates the utility of the plan.

Plan Information

For a selected plan the middle column provides you the following information:

info

The goals you selected.

 info

The properties which are additionally satisfied by the plan.

info

The overall utility of the plan.

info

The properties which are not satisfied by the plan.

If it is not possible to find a plan for the selected goals, the tool will tell you, that the selection of goals is unsolvable.

Task Information

In the left column you can access the task information. This includes a picture of the initial setup and additional information about the contained objects and their constraints.

HOW TO: Property Incompatibilities

Incompatibilities Overview

To open the incompatibilities overview you can click on the Show Incompatibilities Overview button.

Given the constraints (e.g. time, fuel, or energy), it is not possible to achieve all properties. The incompatibilities overview groups the properties that cannot be achieved simultaneously.

Example

For example lists A,B,C and A,D mean it is not possible to achieve properties A, B and C or properties A and D simultaneously.

Filter

You can also filter the property groups with respect to a property. Then only the groups containing the selected property will be displayed.

HOW TO: Ask Questions

Ask Question

As an alternative to the incompatibilities overview, you can also ask specific questions about the currently selected plan. To ask a question you can click the Ask Question button. This opens the question view.

You can ask questions of the structure: "Why not X?". Where X is a property, which is currently not satisfied by the plan.

By pressing the button in the bottom you start the computation of the answer.

Answers

There are different possible answers:

question_answer

If there is no reason, why the property is not satisfied, the system then answers you: "There exists a plan which satisfies the properties in your question."

This means you can add the property in your question to the goals to increase the utility.

question_answer

If it is not possible to satisfy the property in your question at all, the system answers:

"There is no plan which satisfies this property. All global goals and the property in the question can not be satisfied simultaneously."

This means you can not satisfy the property in your questions given the priority property and the fuel, energy or time constraints.

question_answer

The property in your questions can be satisfied, if you give up at least one of the selected goals. The system then tells you which goals you have to give up.

No Plan Possible

If the system can not find a plan for the selected goals you can ask why? The system answers your question by showing you which goals in your selection can not be achieved simultaneously.

HOW TO: Finish

You can leave the task with the Finish button at any time.
